Project Overview.

This is the final assigment of I&C SCI_X460.11: Java Programming II (SUMMER 2016). The assignemnt consisted of implementing a Defect Tracking web application.

Application Basic Features

  • The ability to submit a defect against an application.
  • The ability to update an existing defect.
  • The ability to assign a defect request to a person.
  • The ability to email status information to a user.
  • The ability to view a list of all open defects.

File Structure

The following folder structure can be implemented under your web-server:

  • classes: .class files compiled from .java files
  • jsp: the .jsp files are placed here
  • ROOT: the .html and .css files go here
  • WEB-INF: web.xml needs to be placed here
  • images: place .jpg files here
