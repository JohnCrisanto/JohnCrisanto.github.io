UCI-Extension, Java II - Bug Tracker System Project

Project Overview

In our Java II course, we were instructed to design and build a web application that serves to track application defects. The idea is that our system will be used to track defects against existing applications. It is a simple system used by staff to track bug ticket requests from submission to closure.

Application Basic Features

  • The system allows the user to view all open tickets for all existing applications registered within the system database.
  • The system gives the user the ability to filter which application they want to view existing tickets for.
  • The system allows the user to view the details of an open ticket.
  • The system allows for the submission of a new ticket.
  • The system is configured to notify the assignee of the ticket through email upon updating or creating a ticket.
  • The application utilizes a relational database as a mean for data storage, which allows information to persist between user sessions.
